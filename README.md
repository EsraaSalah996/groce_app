# groce_app
A modern Flutter grocery shopping application built with reusable widgets and clean folder structure.

---

# ✨ Features

*  Login Screen
*  Register Screen
*  home screen
*  product details screen
*  Reusable Widgets
*  Responsive UI
*  Git & GitHub Workflow

* Dart
