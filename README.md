# groce_app
A modern Flutter grocery shopping application built with reusable widgets and clean folder structure.

---

# ✨ Features

*  Login Screen
*  Register Screen
*  Home Screen
*  Product details Screen
*  Explore Screen
*  Cart Screen
*  favourite Screen
*  Account Screen
*  Reusable Widgets
*  Responsive UI
*  Git & GitHub Workflow

---

# 🛠️ Tech Stack

* Flutter
* Dart
